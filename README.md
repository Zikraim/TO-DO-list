Description

A simple and intuitive web app to manage daily tasks. Users can add tasks, mark them as completed, view their status (Pending/Completed), and delete tasks. Tasks are saved in the browser using Local Storage, so they persist even after refreshing the page.

Built with HTML, CSS, and JavaScript, this app features a calm forest-themed design, making task management both functional and visually pleasing.

Features

Add new tasks

Mark tasks as Completed or Pending

Delete tasks

Tasks are saved in Local Storage

Forest-themed design for a clean, natural look

Technologies Used

HTML – for structure

CSS – for styling and Forest theme

JavaScript – for interactivity and Local Storage# TO-DO-list
